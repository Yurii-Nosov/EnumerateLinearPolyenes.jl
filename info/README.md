# Supplementary Materials

This folder contains additional materials for the package `[EnumerateLinearPolyenes]`.

## What's included

- `motivation.md` - motivation for writing an article.
- `terminology.md` — a glossary of terms used in the article.
- `algorithm_overview.md` — a high‑level description of the algorithms.
- `algorithm_general_information.pdf` - general information underlying all algorithms.
- `input_output_guide.md` — documentation of input/output data formats.
